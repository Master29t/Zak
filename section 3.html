Table

Table tag is used to create table in html.
  • tr is used to add table row. You can use multiple tr to add multiple rows
  • td is used to add data in each cell
  • thead is used to create table header
  • th is used to add data in each header cell
  • tbody is used to define the body of the table
  • tfoot is used to create footer of the table

Row 1 Row 1.1
