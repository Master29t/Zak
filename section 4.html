SECTION 4

Cascading Style Sheets - Inline CSS

✤ An inline CSS is used to apply a unique style to a single HTML element.
✤ An inline CSS uses the style attribute of an HTML element.

Internal CSS

✤ An internal CSS is used to define a style for a single HTML page.
✤ An internal CSS is defined in the <head> section of an HTML page, within a <style> element.

External CSS

✤ External style sheets are created in separate documents with a . css extension.
✤ An external style sheet is simply a listing of CSS rules.
✤ It cannot contain HTML tags.
✤ The <link> tag, which goes in the head of an HTML page, is used to link to an external style sheet.
✤ Multiple webpages can use the same link to access the stylesheet.
